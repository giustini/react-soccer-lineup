## Renderizando Equipos

Para mostrar equipos en el campo, pasa los props `homeTeam` y/o `awayTeam`. Cada equipo requiere un objeto `squad` que define las posiciones de los jugadores.

### Local

Con la prop `homeTeam`, el equipo local se renderizará atacando desde el lado izquierdo del campo:

```tsx
import SoccerLineUp, { type Team } from 'react-soccer-lineup';

function App() {
  const homeTeam: Team = {
    squad: {
      gk: { number: 1 },
      df: [{ number: 2 }, { number: 4 }, { number: 5 }, { number: 3 }],
      cm: [{ number: 11 }, { number: 6 }, { number: 8 }, { number: 7 }],
      fw: [{ number: 9 }, { number: 10 }]
    }
  };

  return <SoccerLineUp homeTeam={homeTeam} />;
}
```

      gk: { number: 1 },
      df: [{ number: 2 }, { number: 4 }, { number: 5 }, { number: 3 }],

      cm: [{ number: 10 }, { number: 8 }],
      fw: [{ number: 11 }, { number: 9 }, { number: 7 }]
    }
  };

  return <SoccerLineUp awayTeam={awayTeam} />;
}

### Ambos equipos

Usando ambas props, cada equipo se renderizará en su lado del campo.

```tsx
import SoccerLineUp, { type Team } from 'react-soccer-lineup';

function App() {
  const homeTeam: Team = {
    squad: {
      gk: { number: 1 },
      df: [{ number: 2 }, { number: 4 }, { number: 5 }, { number: 3 }],
      cm: [{ number: 11 }, { number: 6 }, { number: 8 }, { number: 7 }],
      fw: [{ number: 9 }, { number: 10 }]
    }
  };

  const awayTeam: Team = {
      squad: {
        gk: { number: 1 },
        df: [{ number: 2 }, { number: 4 }, { number: 5 }, { number: 3 }],
        cdm: [{ number: 6 }],
        cm: [{ number: 10 }, { number: 8 }],
        fw: [{ number: 11 }, { number: 9 }, { number: 7 }]
      }
    };

  return <SoccerLineUp homeTeam={homeTeam} awayTeam={awayTeam} />;
}
```

## Patrones del Campo

La prop `pattern` añade un patrón visual al campo:

### Líneas

```tsx
import SoccerLineUp from 'react-soccer-lineup';

function App() {
  return <SoccerLineUp pattern="lines" />;
}
```

## Orientación del campo

La prop `orientation` controla si el campo se muestra en horizontal o en vertical.

### Horizontal

La orientación por defecto. Equipo local a la izquierda, visitante a la derecha:

```tsx
import SoccerLineUp from 'react-soccer-lineup';

function App() {
  return <SoccerLineUp orientation="horizontal" />;
}
```

<LiveDemo client:visible orientation="horizontal" />

### Vertical

Disposición vertical. Equipo local abajo, visitante arriba:

```tsx
import SoccerLineUp from 'react-soccer-lineup';

function App() {
  return <SoccerLineUp orientation="vertical" />;
}
```

<LiveDemo client:visible orientation="vertical" />

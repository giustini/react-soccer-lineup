## Team styling

### TeamStyle properties

```typescript
type TeamStyle = {
  color?: string;               // Players fill color
  borderColor?: string;         // Players border color
  numberColor?: string;         // Jersey number text color
  nameColor?: string;           // Player name text color
  nameBackgroundColor?: string; // Name label background
};
```

### Default colors

| Property | Home Team | Away Team |
|----------|-----------|-----------|
| `color` | `#ffffff` | `#333333` |
| `borderColor` | `#ffffff` | `#333333` |
| `numberColor` | `#333333` | `#ffffff` |
| `nameColor` | `#333333` | `#ffffff` |
| `nameBackgroundColor` | transparent | transparent |

### Custom team colors

```tsx
const team: Team = {
  squad: { /* ... */ },
  style: {
    color: '#a71f28',
    borderColor: '#ffffff',
    numberColor: '#ffffff',
    nameColor: '#ffffff',
    nameBackgroundColor: '#33333cc'
  }
};
```

## Style Inheritance

Styles cascade in this order (later overrides earlier):

1. **Default colors** (home: white, away: dark)
2. **Team style** (`team.style`)
3. **Player style** (`player.style`)

```tsx
// Player inherits team style, but overrides color
const team: Team = {
  squad: {
    gk: { number: 1 },
    df: [
      { number: 2 },  // Uses team style
      { number: 4, style: { color: '#ffd700' } },  // Gold, rest from team
      { number: 5 },  // Uses team style
      { number: 3 }   // Uses team style
    ],
    // ...
  },
  style: {
    color: '#c8102e',
    borderColor: '#ffffff'
  }
};
```

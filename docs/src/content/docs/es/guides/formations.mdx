En esta página se describe el comportamiento interno del componente principal, y cómo se pueden implementar las formaciones más comunes de fútbol, así como cualquier formación deseada con cualquier número de jugadores.

## Entendiendo las formaciones

Las formaciones se definen distribuyendo jugadores a través de líneas de posición:

- `gk` - Portero (1 jugador)
- `df` - Defensas (línea trasera)
- `cdm` - Mediocampistas Defensivos Centrales
- `cm` - Mediocampistas Centrales
- `cam` - Mediocampistas Ofensivos Centrales
- `fw` - Delanteros (línea delantera)

### Posicionamiento de las líneas

Todas las líneas de posición son opcionales; se pueden proporcionar, o no. Y el número de jugadores de cada línea es totalmente libre, excepto en la línea del portero que sólo puede contener un jugador.

Esto significa que si una línea no se proporciona no se tendra en cuenta para renderizar el equipo. Pero, si se proporciona como un array vacío (o como `null` en el caso del portero), la línea se renderizará sin jugadores.

Por ejemplo, es posible renderizar un equipo sin portero:

```tsx
const squad: Squad = {
  gk: null,
  df: [{ number: 2 }, { number: 4 }, { number: 5 }, { number: 3 }],
  cm: [{ number: 7 }, { number: 6 }, { number: 8 }, { number: 11 }],
  fw: [{ number: 9 }, { number: 10 }]
};
```

### Posicionamiento de los jugadores

Los jugadores de cada línea se distribuyen de manera uniforme sobre ella. Además, se puede utilizar `null` para ocupar el espacio de un jugador sin que se renderice nada, lo que añade mucha flexibilidad a la hora de configurar las formaciones.

Por ejemplo, una formación con laterales adelantados, mediapunta caído a una banda y delanteros separados:

```tsx
const squad: Squad = {
  gk: { number: 1 },
  df: [null, { number: 4 }, { number: 5 }, null],
  cdm: [{ number: 2 }, null, null, null, { number: 3 }],
  cm: [{ number: 6 }, { number: 8 }, { number: 10 }],
  cam: [null, null, { number: 11 }, null],
  fw: [{ number: 9 },  null, { number: 7 }]
};
```

Para afinar todavía más las formaciones, se puede añadir un desplazamiento adicional mediante la propiedad [`offset`](/es/guides/players/#desplazamiento-de-posición) del jugador.

```tsx
const squad: Squad = {
  gk: { number: 1 },
  df: [{ number: 2, offset: { x: 8 } }, { number: 4 }, { number: 5 }, null],
  cdm: [null, null, null, null, { number: 3 }],
  cm: [{ number: 6, offset: { x: 8 } }, { number: 8, offset: { x: -8 } }, { number: 10 }],
  cam: [null, null, { number: 11 }, null],
  fw: [{ number: 9, offset: { x: 6, y: -15 } },  null, { number: 7 }]
};
```

  cm: [{ number: 5 }],
  fw: [{ number: 6 }, { number: 7 }]
};

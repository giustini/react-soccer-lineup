## Propiedades del jugador

Cada jugador puede tener las siguientes propiedades:

```typescript
type Player = {
  name?: string;
  number?: number;
  style?: PlayerStyle;
  offset?: PlayerOffset;
  onClick?(): void;
};
```

### Jugador básico

Para renderizar un jugador, como mínimo se necesita un objeto vacío:

```tsx
const player: Player = { };
```

### Jugador con número

Opcionalmente, puedes especificar el número de la camiseta:

```tsx
const player: Player = { number: 7 };
```

### Jugador con nombre

O añadir un nombre para mostrar debajo del jugador:

```tsx
const player: Player = {
  name: 'C. Ronaldo',
  number: 7
};
```

## Estilo individual del jugador

Sobrescribe los estilos del equipo para jugadores específicos:

```tsx
const captain: Player = {
  name: 'Captain',
  number: 7,
  style: {
    color: '#ffd700', // Gold background
    numberColor: '#333',
    nameColor: '#333',
    nameBackgroundColor: '#ffd700'
  }
};
```

## Desplazamiento de Posición

Ajusta las posiciones de los jugadores con `offset`:

| Offset | Direction |
|--------|-----------|
| `x` positive | Right |
| `x` negative | Left |
| `y` positive | Down |
| `y` negative | Up |

```tsx
const player: Player = {
  name: 'C. Ronaldo',
  number: 7
  offset: {
    x: 55,   // Move 20px right
    y: 40   // Move 10px down
  }
};
```

## Click handler

Añade interactividad con `onClick`:

```tsx
const player: Player = {
  name: 'C. Ronaldo',
  number: 7
  onClick: () => {
    console.log('Player clicked!');
    // Show player stats, open modal, etc.
  }
};
```

Cuando se proporciona `onClick`, el jugador muestra un cursor de puntero al pasar el mouse.
